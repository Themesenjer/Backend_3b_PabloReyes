# Comandos GitHub

# Inicializar el repositorio local
```
git init
```
- Referencia del repositorio local con el repositorio remoto (GITHUB)
````
git remote add origin https://github.com/Themesenjer/Backend_3b_PabloReyes.git
````

# Validar referencia
````
git remote -v
````
